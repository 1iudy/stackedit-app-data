# **FeNiCrCoAl合金分子动力学计算**
## **atomsk生成高熵合金超胞模型**
### **1.等原子比超胞构建与晶格常数弛豫**
根据文章内容，首先生成10×10×10的Fe超胞结构,FCC结构，共有4000个原子
```
atomsk --create fcc 3.60 Fe -duplicate 10 10 10 Fe_supercell.xsf 
```

按照等原子比（1：1：1：1：1），各类原子各800个，分别用800个Ni、Cr、Co、Al原子对Fe原子进行替代
```
atomsk Fe_supercell.xsf  -select random 800 Fe -sub Fe Ni -select random 800 Fe -sub Fe Cr -select random 800 Fe -sub Fe Co -select random 800 Fe -sub Fe Al equiatomic.xsf
```

也可以按照原子比例进行替代，由于每次替代之后Fe原子的数量改变，因此各元素替换的比例不同
```
atomsk Fe_supercell.xsf  -select random 20% Fe -sub Fe Ni -select random 25% Fe -sub Fe Cr -select random 33.3333% Fe -sub Fe Co -select random 50% Fe -sub Fe Al equiatomic_1.xsf
```

使用ovito查看生成的超胞结构：![输入图片说明](/imgs/2026-06-30/B4XUvbmywvjfySpj.png)
（配合使用Select particle type和Delete selection修改器可以观察单一种类原子分布情况）
超胞弛豫in文件如下
```
units metal
dimension 3
boundary p p p

atom_style atomic
read_data equiatomic.lmp

pair_style eam/alloy
pair_coeff * * ../../FeNiCrCoAl-heaweight.setfl Fe Ni Cr Co Al
  
#设置近邻列表，金属体系通常2.0-3.0 Ångström，
neighbor 2.0 bin 
neigh_modify every 1 delay 0 check yes #

#设置输出 
reset_timestep 0	#重置时间步
thermo 100		#每100步输出一次
thermo_style custom step pe etotal press pxx pyy pzz lx ly lz		#输出格式
  
#最小化
min_style cg
fix 1 all box/relax iso 0.0 vmax 0.01
minimize 1.0e-12 1.0e-12 10000 100000
unfix 1

write_data relaxed_equiatomic.lmp
```
计算结果为3.626Å，与文献相符

### **2.元素含量与平衡晶格常数曲线绘制**
首先生成Fe超胞结构
```
atomsk --create fcc 3.60 Fe -duplicate 20 20 20 Fe_supercell.xsf
```
超胞共32000个原子，需要构建的高熵合金超胞Fe含量至少为35 at%.，除Fe以外，其余元素含量在11.75–16.25 at%.之间，对应的原子数为3760-5200之间，含量间隔为0.5 at%.，对应的原子数为160。
生成处Fe以外元素等原子比的命令如下
```
atomsk Fe_supercell.xsf  -select random 5200 Fe -sub Fe Ni -select random 5200 Fe -sub Fe Cr -select random 5200 Fe -sub Fe Co -select random 5200 Fe -sub Fe Al FeNiCrCoAl.xsf
```
后续对于需要改变的元素替换原子数依次减160即可，对应原子数分别为
5200	5040	4880	4720	4560	4400	4240	4080	3920	3760

>注意：以上方式使用atomsk生成的超胞.lmp文件中原子顺序是随机的，in文件中的原子顺序需要调整（random命令导致）

复制in文件和任务提交文件：
```
echo 1 2 3 4 5 6 7 8 9 | xargs -n 1 cp 0/in.lmp 0/job.slurm
```

### **3.FeNiCrCoAl与Fe35NiCrCoAl单向等轴拉伸及压缩**
首先生成Fe超胞结构，按照文献，X、Y和Z方向分别设置为[112]、[-110]和[-1−11]，模型单元格大小为20x20x12，包含28,800个原子
```
atomsk --create fcc 3.625 Fe orient [112] [-110] [-1-11] -duplicate 20 20 12 Fe_supercell.xsf
```
