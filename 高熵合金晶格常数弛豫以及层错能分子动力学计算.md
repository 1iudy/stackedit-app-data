# **FeNiCrCoAl合金分子动力学计算**
## **atomsk生成高熵合金超胞模型**
### **1.等原子比超胞构建与晶格常数弛豫**
根据文章内容，首先生成10×10×10的Fe超胞结构,FCC结构，共有4000个原子
```
atomsk --create fcc 3.60 Fe -duplicate 10 10 10 Fe_supercell.xsf 
```

按照等原子比（1：1：1：1：1），各类原子各800个，分别用800个Ni、Cr、Co、Al原子对Fe原子进行替代
```
atomsk Fe_supercell.xsf  -select random 800 Fe -sub Fe Ni -select random 800 Fe -sub Fe Cr -select random 800 Fe -sub Fe Co -select random 800 Fe -sub Fe Al equiatomic.xsf
```

也可以按照原子比例进行替代，由于每次替代之后Fe原子的数量改变，因此各元素替换的比例不同
```
atomsk Fe_supercell.xsf  -select random 20% Fe -sub Fe Ni -select random 25% Fe -sub Fe Cr -select random 33.3333% Fe -sub Fe Co -select random 50% Fe -sub Fe Al equiatomic_1.xsf
```

使用ovito查看生成的超胞结构：![输入图片说明](/imgs/2026-06-30/B4XUvbmywvjfySpj.png)
（配合使用Select particle type和Delete selection修改器可以观察单一种类原子分布情况）
超胞弛豫in文件如下
```
units metal
dimension 3
boundary p p p

atom_style atomic
read_data equiatomic.lmp

pair_style eam/alloy
pair_coeff * * ../../FeNiCrCoAl-heaweight.setfl Fe Ni Cr Co Al
  
#设置近邻列表，金属体系通常2.0-3.0 Ångström，
neighbor 2.0 bin 
neigh_modify every 1 delay 0 check yes #

#设置输出 
reset_timestep 0	#重置时间步
thermo 100		#每100步输出一次
thermo_style custom step pe etotal press pxx pyy pzz lx ly lz		#输出格式
  
#最小化
min_style cg
fix 1 all box/relax iso 0.0 vmax 0.01
minimize 1.0e-12 1.0e-12 10000 100000
unfix 1

write_data relaxed_equiatomic.lmp
```
计算结果为3.626Å，与文献相符

### **2.元素含量与平衡晶格常数曲线绘制**
首先生成Fe超胞结构
```
atomsk --create fcc 3.60 Fe -duplicate 20 20 20 Fe_supercell.xsf
```
超胞共32000个原子，需要构建的高熵合金超胞Fe含量至少为35 at%.，除Fe以外，其余元素含量在11.75–16.25 at%.之间，对应的原子数为3760-5200之间，含量间隔为0.5 at%.，对应的原子数为160。
生成处Fe以外元素等原子比的命令如下
```
atomsk Fe_supercell.xsf  -select random 5200 Fe -sub Fe Ni -select random 5200 Fe -sub Fe Cr -select random 5200 Fe -sub Fe Co -select random 5200 Fe -sub Fe Al FeNiCrCoAl.xsf
```
后续对于需要改变的元素替换原子数依次减160即可，对应原子数分别为
5200	5040	4880	4720	4560	4400	4240	4080	3920	3760

>注意：以上方式使用atomsk生成的超胞.lmp文件中原子顺序是随机的，in文件中的原子顺序需要调整（random命令导致）

计算晶格常数进行的超胞弛豫与上一节相同，输入文件的设置相同
复制in文件和任务提交文件：
```
echo 1 2 3 4 5 6 7 8 9 | xargs -n 1 cp 0/in.lmp 0/job.slurm
```

### **3.FeNiCrCoAl与Fe35NiCrCoAl单向等轴拉伸及压缩**
首先生成Fe超胞结构，按照文献，X、Y和Z方向分别设置为[112]、[-110]和[-1−11]，模型单元格大小为20x20x12，包含28,800个原子
```
atomsk --create fcc 3.625 Fe orient [112] [-110] [-1-11] -duplicate 20 20 12 Fe_supercell.xsf
```
对于等原子比FeNiCrCoAl合金，各使用5760个Ni、Cr、Co、Al随机替代Fe原子：
```
atomsk Fe_supercell.xsf -select random 5760 Fe -sub Fe Ni -select random 5760 Fe -sub Fe Cr -select random 5760 Fe -sub Fe Co -select random 5760 Fe -sub Fe Al FeNiCrCoAl.xsf
```

![输入图片说明](/imgs/2026-07-01/8aKn6TLE3MqRXpCD.png)
对于Fe35NiCrCoAl合金，各使用4680个Ni、Cr、Co、Al随机替代Fe原子：
```
atomsk Fe_supercell.xsf -select random 4680 Fe -sub Fe Ni -select random 4680 Fe -sub Fe Cr -select random 4680 Fe -sub Fe Co -select random 4680 Fe -sub Fe Al Fe35NiCrCoAl.xsf
```
单向拉伸in文件如下，压缩模拟输入文件只需将施加的单位应变改为负值即可
```
units metal
dimension 3
boundary p p p
  
atom_style atomic
read_data FeNiCrCoAl.lmp
  
pair_style eam/alloy
pair_coeff * * ../../FeNiCrCoAl-heaweight.setfl Co Cr Ni Al Fe
  
#计算量定义，用于后续分析
compute csym all centro/atom fcc #计算每个原子的中心对称参数（识别FCC结构完整性）
compute peratom all pe/atom #计算每个原子的势能(分析能量分布)
  
#1.平衡模拟（消除初始应力，使体系达到稳定状态）
reset_timestep 0
timestep 0.001 #时间步长设置为0.001ps，即1fs，metal单位制下基本单位为ps
  
velocity all create 300.0 12345 mom yes rot no #初始化速度，温度为300K，随机种子为12345，保留动量（mom yes），消除角动量（rot yes），速度分布为默认（dist）
#NPT系综设置：temp 300 300 1：温度保持300K，时间常数（热浴阻尼系数）为1ps
# iso 0.0 0.0 1：各向同性压力为0，时间常数（压强阻尼系数）为1ps
# drag 1：原子运动阻尼系数为1，防止体系过度振荡
fix 1 all npt temp 300.0 300.0 1.0 iso 0.0 0.0 1.0 drag 1.0
  
#输出热力学信息，每1000步输出一次，输出内容包括：步数、盒子尺寸（x/y/z）、总压力、压力张量分量、势能、温度
thermo 1000
thermo_style custom step lx ly lz press pxx pyy pzz pe temp
  
run 20000 #模拟20000步，时间总长为20ps
unfix 1
  
variable tmp1 equal lx
variable L0 equal ${tmp1}
print "The initial Length L0 = ${L0}"
  
#2.单向拉伸模拟（沿x方向拉伸，y和z方向保持零应力）
reset_timestep 0
thermo_style custom step press pxx pyy pzz
run 0
#重新施加NPT系综，保持y和z方向零应力，x方向施加拉伸应变率
fix 1 all npt temp 300.0 300.0 1.0 y 0.0 0.0 1.0 z 0.0 0.0 1.0 drag 1.0
#设置拉伸应变速率
variable strate0 equal 1.0e10 #lammps模拟高应变率拉伸，单位为1/s
variable strate equal "v_strate0/1.0e12" #将应变速率转换为lammps时间单位（1/ps)
  
#组1沿x方向进行拉伸变形：
# deform 1 x：沿x方向施加拉伸变形，施加对象为组1
# erate ${strate}：施加的应变速率为1.0e10/s
# units box：应变速率单位为盒子尺寸单位
# remap x：在拉伸过程中，原子坐标随盒子尺寸变化而重新映射，保持原子间距随盒子拉伸而变化
fix 2 all deform 1 x erate ${strate} units box remap x
  
#拉伸数据输出
variable tmp2 equal "(lx - v_L0)/v_L0" #定义应变变量，计算当前长度与初始长度的比值
variable strain equal "v_tmp2" #将应变变量赋值给strain
variable p1 equal "-pxx/10000" #定义应力变量，计算x方向的应力（单位为GPa）
variable p2 equal "-pyy/10000" #定义应力变量，计算y方向的应力（单位为GPa）
variable p3 equal "-pzz/10000" #定义应力变量，计算z方向的应力（单位为GPa）
  
fix def1 all print 100 "${strain} ${p1} ${p2} ${p3}" file tensile_stress_strain.txt screen no
#输出轨迹信息，每100步输出一次，输出内容包括：应变、x/y/z方向应力
dump trace all custom 100 tensile_FeNiCrCoAl.lammpstrj id type x y z c_csym c_peratom
  
#热力学信息输出
thermo 1000
thermo_style custom step temp pe ke press v_tmp2 pxx pyy pzz
run 30000 #模拟30000步，时间总长为30ps
unfix 1
unfix 2
unfix def1
  
print "Simulation completed. "
```
